#  VUE 3.x 项目介绍 （EWShop）

## 项目说明

后台开发： PHP+Laravel+MySQL + Redis + Restrestful api接口规范

前台模板： Vue3.x + Composition API + Vant组件库

后台管理： React

接口文档 ：https://www.showdoc.com.cn/1207745568269674
